## 项目简介

基于 [Astro Theme Pure](https://github.com/cworld1/astro-theme-pure) 模板搭建的个人网站，用于展示博客文章、项目作品和个人信息。

## 技术栈

- **Framework**: Astro 5
- **Theme**: Astro Theme Pure v1.3.4
- **Styling**: UnoCSS
- **Hosting**: Cloudflare Pages
- **AI Assistant**: Claude Code

## 功能

- 博客文章发布与管理
- 项目作品展示
- 公众号文章索引
- 全文搜索（Pagefind）
- RSS 订阅
- 深色/浅色模式切换
